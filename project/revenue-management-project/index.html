Airlines Revenue Management Decision Support Tool

Jun 7, 2017

This is a term project for Decision Support Systems course. I utilized advanced level abilities including vlookups and Macros in MS Excel to provide an interactive user interface for revenue management problem of an airlines company.

First, mathematical model is created for maximizing profit by finding number of seats to be allocated for cabin classes in each flight. Then, I performed a sensitivity analysis under different scenarios using What-If Analysis tool. Additional functioalities include helping users update data and finding the best route based on different parameters.
